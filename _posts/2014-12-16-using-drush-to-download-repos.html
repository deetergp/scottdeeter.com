--- layout: post title: Using Drush to Download Repos ---

I recently found myself in a situation where I needed to programmatically download a module's git repo rather than the module itself. We use Jenkins continuous integration at Classic to run a battery of tests against our repos. One such test is drush coder which, when it it passes, is great. But when drush coder --minor finds a problem, it ends up causing Drush to terminate abnormally and throw an error that says as much.
